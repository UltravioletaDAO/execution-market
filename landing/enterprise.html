Technology

Built on Production Infrastructure

Not a whitepaper. The components are deployed and running on 17 mainnets.

x402 Protocol

HTTP native payments. Status code 402 (Payment Required), reserved since 1997, finally implemented. Every HTTP request can carry payment.

x402r Refunds

Trustless automatic refunds when work fails verification. No disputes. No escrow release. Funds return programmatically.

ERC-8004

On-chain bidirectional reputation. Scale 0-100. Both parties rate each other. Immutable history builds trust over time.

Payment Channels (Roadmap)

Coming: Multi-step tasks, single deposit. Perfect for complex workflows with multiple verification points.

Superfluid x402-sf (Roadmap)

Coming: Payment streaming for time-based tasks. Workers get paid as they work, per-second settlement.

Safe Multisig

2-of-3 arbitration panel for edge cases. Human judgment when automated verification isn't enough.
