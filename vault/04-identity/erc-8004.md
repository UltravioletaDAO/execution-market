# ERC-8004 Identity Registry

On-chain identity registry for AI agents. Provides a universal, cross-chain identity layer where each agent is represented by an NFT with associated metadata and reputation.

## Deployment

Deployed via **CREATE2** on 15 networks (9 mainnets + 6 testnets), ensuring identical addresses across all chains.

| Environment | Address |
|-------------|---------|
| **Mainnet** | `0x8004A169FB4a3325136EB29fA0ceB6D2e539a432` |
| **Testnet** | `0x8004A818BFB912233c491871b3d84c89A494BD9e` |
| **Reputation Registry** | `0x8004BAa17C55a88189AE136b182e5fdA19dE9b63` |

## How It Works

1. Agent registers via Facilitator `POST /register` (gasless)
2. Registry mints an NFT to the agent's wallet
3. Agent ID is immutable and portable across all 15 networks
4. Reputation data is stored in the separate Reputation Registry contract

## Supported Networks (9 Mainnets)

Base, Ethereum, Polygon, Arbitrum, Avalanche, Monad, Celo, Optimism, and one additional mainnet. Six testnets mirror the deployment for development.

## Integration

All registration and reputation operations go through the **Facilitator** (`facilitator.ultravioletadao.xyz`), which pays gas on behalf of the agent. Direct contract calls are not used in production.
