## Recommended pilot sequencing

1. **Bounded Local Count** — lowest authority and teaches method/uncertainty discipline.
2. **Visible Asset State Snapshot** — strong operational value if no-touch/no-repair boundaries stay explicit.
3. **Pre-Event Blocker Check** — valuable but time pressure increases overclaim risk, so keep limitation language adjacent.
4. **Handoff Attempt Proof** — commercially useful but requires redaction/delivery-path approval before exposure.
5. **Compliance Hold Review** — keep label-only until allowed text and delivery path exist.
6. **Counter Reality Check** — strongest City Ops wedge, but it should wait for the separate operator answer/hold path before exposure.

This is sequencing for internal planning only. It is not a launch order, approval order, sales promise, route order, dispatch order, pricing order, or customer priority.
